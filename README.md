## Installation

```
npm i avishka-ui
```
or
```
yarn add avishka-ui
```

## Usage

```js
// src/App.js
import { Button, Badge, Typography, IconButton, Image, Alert } from "avishka-ui";
```

### for next.js
```js
// pages/_app.js
import 'avishka-ui/dist/style/main.css'
```
now you can use

```js
// pages/index.js
import { Button, Badge, Typography, IconButton, Image, Alert } from "avishka-ui/next";
```
